# Reclaude — Frontend

React 19 + TypeScript + Vite + TailwindCSS v4 + shadcn/ui

## Setup

```sh
bun install
```

## Development

```sh
bun run dev
```

Open http://localhost:3000 (API requests to `/api` are proxied to the backend on :3001).

## Linting & Formatting

```sh
bun run lint        # Check for issues
bun run lint:fix    # Auto-fix issues
bun run format      # Format with Prettier
bun run typecheck   # Type check
```
